Question 1

What is computer programming, and why is it important in today's technology-driven world?

Answer:

Computer programming is the process of writing instructions that tell a computer how to perform specific tasks. It is important because it allows people to create software, automate repetitive tasks, solve problems, and power many of the technologies we use every day.

Question 2

Briefly describe what Python is and one reason it is a popular programming language for beginners.

Answer:

Python is a high-level programming language used to build applications, automate tasks, analyze data, and develop artificial intelligence solutions. It is popular for beginners because its syntax is simple, easy to read, and easier to learn than many other programming languages.

Question 3

What do you need (software/tools) to start writing and running Python programs on your own computer?

Answer:

To start writing and running Python programs, you need to install Python on your computer and use a code editor such as Visual Studio Code. You also need the Python interpreter and a terminal or command prompt to run your Python files.
